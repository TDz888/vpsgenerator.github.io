🚀 VPS Generator Pro

Windows Server + Tailscale + noVNC | Tạo VPS miễn phí
📌 Hướng dẫn:
1. Tạo token tại: github.com/settings/tokens
2. Chọn quyền: repo, workflow
3. Dán token vào ô trên và bấm tạo
4. Đợi 2-3 phút, sau đó dùng 1 trong 2 cách bên dưới:

🔗 Cách 1 - Tailscale (Khuyên dùng):
- Cài Tailscale: tailscale.com/download
- Đăng nhập cùng tài khoản với VM
- Lấy IP Tailscale từ GitHub Actions logs
- Dùng Remote Desktop kết nối đến IP đó

🌐 Cách 2 - noVNC (Web):
- Click vào link noVNC sau khi tạo
- Nhập mật khẩu: VPS@123456
- Bắt đầu sử dụng Windows
