📖 Hướng dẫn lấy thông tin xác thực

1
🔑 Lấy GitHub Token
Truy cập github.com/settings/tokens
→ Chọn "Generate new token (classic)" → Tích chọn quyền repo và workflow
→ Copy token bắt đầu bằng github_pat_
2
🔗 Lấy Tailscale Auth Key
Truy cập login.tailscale.com/admin/authkeys
→ Chọn "Generate auth key" → Chọn quyền Reusable
→ Copy key bắt đầu bằng tskey-
3
🚀 Tạo VPS
Dán token và key vào ô bên dưới → Nhấn "Khởi tạo VM" → Đợi 2-3 phút
