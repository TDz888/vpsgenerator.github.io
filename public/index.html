🚀 VPS Generator

Tạo VPS miễn phí từ GitHub Actions
📌 Hướng dẫn:
1. Tạo token tại: github.com/settings/tokens
2. Chọn quyền: repo, workflow
3. Dán token vào ô trên và bấm tạo
4. VPS sẽ chạy trong GitHub Actions

✅ Sau khi tạo bạn sẽ có:
  • 📁 Repository chứa workflow
  • ⚙️ GitHub Actions chạy liên tục
  • 🔄 Tự động restart mỗi 10 phút
